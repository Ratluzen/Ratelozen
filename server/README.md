# Ratelozn Backend Server

هذا السيرفر يمثل البنية التحتية الكاملة لتطبيق Ratelozn Services.

## المتطلبات
1. **Node.js** (مثبت على الجهاز).
2. **MongoDB** (يجب أن يكون لديك قاعدة بيانات محلية أو رابط MongoDB Atlas).

## خطوات التشغيل

1. افتح نافذة Terminal جديدة وتوجه لمجلد السيرفر:
   ```bash
   cd server
   ```

2. قم بتثبيت المكتبات:
   ```bash
   npm install
   ```

3. قم بإنشاء ملف `.env` وانسخ محتويات `.env.example` إليه، وعدل رابط قاعدة البيانات إذا لزم الأمر.

4. تشغيل السيرفر:
   ```bash
   npm start
   ```
   أو لوضع المطور (مع إعادة التشغيل التلقائي):
   ```bash
   npm run dev
   ```

## الميزات المتوفرة (API Endpoints)
- **المستخدمين:** تسجيل، دخول، ملف شخصي، إدارة الرصيد.
- **المنتجات:** إدارة كاملة (إضافة، تعديل، حذف) مع دعم الدول والكميات.
- **الطلبات:** إنشاء طلب، التسليم التلقائي (Auto-Delivery)، الإلغاء، التنفيذ اليدوي.
- **المخزون:** إضافة أكواد، ربط الأكواد بالمنتجات والدول.
- **المحتوى:** إدارة البانرات، الإشعارات، الفئات.
- **الإعدادات:** العملات، الشروط، الروابط.

تم بناء السيرفر ليكون جاهزاً للربط مع `App.tsx` في المستقبل عبر `fetch` أو `axios`.
